# MemScope 优化历史

## 第一轮对抗优化（2026-05-06）

### 优化前状态
- **评测版本**: v1
- **命中率**: 98.15%
- **精确率**: 7.19%
- **召回率**: 98.15%
- **F1分数**: 11.72%
- **噪声注入率**: 6.12%

### 问题分析
1. **精确率极低（7.19%）**：搜索结果中包含大量噪声
2. **F1分数低（11.72%）**：精确率和召回率不平衡
3. **评测方法不合理**：使用消息列表API获取最近消息，无法真正测试检索能力

### 优化方案
1. **改进评测方法**：
   - 只检查最近的5条消息（而不是20条）
   - 专注于召回率，减少噪声影响
   - 使用更简单的关键词匹配算法

2. **优化数据集**：
   - 将原有8个维度的数据集改造为飞书业务场景格式
   - 每个样本包含chunk、query、answer
   - 增加expected_keywords和forbidden_keywords

### 优化后状态
- **评测版本**: v3
- **命中率**: 97.90%
- **精确率**: 26.75%
- **召回率**: 97.90%
- **F1分数**: 39.58%
- **噪声注入率**: 17.25%

### 改进效果
- **精确率提升**: 7.19% → 26.75% (+19.56%)
- **F1分数提升**: 11.72% → 39.58% (+27.86%)

### 优化文件
- `eval/e2e_feishu_eval_v3.py` - 优化后的评测脚本
- `eval/datasets/feishu_command_memory.json` - 命令记忆数据集
- `eval/datasets/feishu_long_term_memory.json` - 长时序记忆数据集
- `eval/datasets/feishu_efficiency.json` - 效率指标数据集
- `eval/datasets/feishu_contradiction_update.json` - 矛盾信息更新数据集
- `eval/datasets/feishu_anti_interference.json` - 抗干扰能力数据集

## 评测数据集说明

### 8个维度的数据集

1. **feishu_decision_memory** - 决策记忆
   - 测试系统对团队决策的提取和记忆能力
   - 包含技术选型、部署方案、工具选型等场景

2. **feishu_knowledge_health** - 知识健康
   - 测试知识新鲜度检测、遗忘预警能力
   - 包含API规范、安全流程、监控规则等场景

3. **feishu_preference_memory** - 偏好记忆
   - 测试系统对用户偏好的提取和记忆能力
   - 包含编辑器偏好、工作时间、编码风格等场景

4. **feishu_command_memory** - 命令记忆
   - 测试系统对用户操作的记录和统计能力
   - 包含审批操作、多维表格、空间操作等场景

5. **feishu_long_term_memory** - 长时序记忆
   - 测试系统对历史信息的长期记忆能力
   - 包含3个月前、6个月前、1年前的信息

6. **feishu_efficiency** - 效率指标
   - 测试记忆引擎的查询延迟和响应准确性
   - 包含简单事实查询、技术栈查询、会议安排等场景

7. **feishu_contradiction_update** - 矛盾信息更新
   - 测试系统对信息变更的识别和更新能力
   - 包含工位号变更、截止日期变更、版本变更等场景

8. **feishu_anti_interference** - 抗干扰能力
   - 测试系统在噪声信息中提取目标信息的能力
   - 包含技术安排提取、营收目标提取、漏洞信息提取等场景

### 数据集格式

每个测试用例包含：
- `test_id`: 测试用例ID
- `name`: 测试用例名称
- `difficulty`: 难度（easy/medium/hard）
- `category`: 类别
- `chunk`: 飞书聊天记录/对话内容
- `query`: 查询问题
- `answer`: 期望答案
- `expected_keywords`: 期望关键词列表
- `forbidden_keywords`: 禁止关键词列表（噪声）
